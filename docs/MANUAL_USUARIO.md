# Manual de Usuario

## Requisitos

- Node.js instalado
- Un navegador moderno

## Ejecución

1. Ingresar a [backend/package.json](/C:/Users/lopez/Documents/New%20project/OLC1_Proyecto2_TuCarnet/backend/package.json)
2. Ejecutar `npm install`
3. Ejecutar `npm run build`
4. Ejecutar `node dist/index.js`
5. Abrir [frontend/index.html](/C:/Users/lopez/Documents/New%20project/OLC1_Proyecto2_TuCarnet/frontend/index.html)

## Uso

1. Crear un archivo nuevo o abrir uno con extensión `.gst`
2. Escribir el código GoScript en el editor
3. Presionar `Ejecutar`
4. Revisar:
   - Consola
   - Reporte de errores
   - Tabla de símbolos
   - AST en formato DOT

## Funcionalidades disponibles

- Crear archivo
- Abrir archivo `.gst`
- Guardar archivo `.gst`
- Ejecutar código
- Visualizar reportes del proyecto
